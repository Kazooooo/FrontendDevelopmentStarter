# FrontendDevelopmentStarter

## Environment
node v7.2.0  
npm 3.10.9 (yarn v0.17.9)  

## Install dependency
```
yarn install
```
If you use npm
```
npm install
```

## Watching JS SCSS and EJS Compiling
```
yarn run gulp
```

## Build only JS files using Webpack
```
yarn run build
```

deploy without sourcemaps and creating minified files
```
yarn run deploy
```
