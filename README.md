# FrontendDevelopmentStarter

## Environment
node v7.2.0  
npm 3.10.9 (yarn v0.17.9)  

## Install dependency
```
npm install
```
If you want to use yarn
```
yarn
```

## Watching scss and ejs Compiling
```
gulp
```

## Build bundle.js using Webpack
```
npm run build
```
watching task
```
npm run watch
```
deploy without sourcemaps and creating minified files
```
npm run deploy
```
